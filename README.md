## Features & Use Cases

- Encode files to MP4 format
- Decode encoded MP4 files back to their original files
- Extension bypass

<br>

## Installation

```bash
npm install -g fideo
```

<br>

## Usage

File to Video:

```bash
f2v <input_file> [-o <output_video>] [-r <even_resolution>]
```

Video to File:

```bash
v2f <input_video> [-o <output_file>]
```
